Welcome to FitFusion
===================================

FitFusion is an app designed for students, to match with other gym goers based on their preferences (e.g. skills).
The system includes:
- a matchmaking page to allow users to match with others
- a messaging system to allow users to communicate with each other
- an events page to view/ join new events
- a register/ login page to allow  the user to register/login/ log out
- a profile page to allow the user to edit their profile
- a preferences page to allow the user to edit their preferences
